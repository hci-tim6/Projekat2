Event information managment:

You can update each field of an existing event or create a new one.
Upon clicking Submit, the submited info will be validated. Upon clicking Cancel, every change will be discarted.

To pass validation each of the necessary fields must be filled and the chosen label must be unique.
Failing the validation will not allow you to change or create a new event.

Event fields

  • Label, must be unique *
  • Name*
  • Description
  • Image, if not chosen it will use the same image as the type
  • Type*
  • Tags, where you can choose one or more

  • Allowed alchohol consumption*
  • Handicap accessability*
  • Prohibition of smoking*
  • Location*
  • Target audience*
  • Price*
Fields marked with * are considered necessary.
