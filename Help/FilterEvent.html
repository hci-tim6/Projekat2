Event filtering:

You can filter existing events by their:

  • Alcohol consumption, where the options are:
    • Allowed to use
    • Prohibited to use
    • Sold there
  • Handicap accessability, where the options are:
    • Accessible
    • Inaccessible
  • Prohibition of smoking, where the options are:
    • Allowed
    • Prohibited
  • Location, where the options are:
    • Indoors
    • Outdoors
  • Target audience, where the options are:
    • Children
    • Adults
    • Elderly
  • Price, where the options are:
    • Free
    • Low
    • Medium
    • High
  • Accessibility on the map, where the options are:
    • Show the events on the current map
    • Show the events not on the current map
    • Every filter can be used in combination with any other.
