Type information managment:

You can update each field of an existing type or create a new one.
Upon clicking Submit, the submited info will be validated. Upon clicking Cancel, every change will be discarted.

To pass validation each of the necessary fields must be filled and the chosen label must be unique.
Failing the validation will not allow you to change or create a new type.

Event fields

  • Label, must be unique *
  • Name*
  • Description
  • Image*
Fields marked with * are considered necessary.
